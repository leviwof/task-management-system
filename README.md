# Task Management System

Full Stack Task Management System built as a take-home assessment.

## Tech Stack
Frontend:
- React
- Axios
- Context API
- CSS

Backend:
- Node.js
- Express
- MongoDB
- JWT Authentication

## Features
- User Registration & Login
- JWT-based Authentication
- Create, Update, Delete Tasks
- Task Filtering by Status & Priority
- Protected Routes
- Secure API Access

## Setup Instructions

### Backend
```bash
cd tms_backend
npm install
npm run dev
